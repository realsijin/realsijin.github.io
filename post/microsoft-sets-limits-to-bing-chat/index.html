【资讯】微软已经为 Bing Chat 设置对话次数上限

作者：realsijin | 发表于： 2023-02-18

微软已经在 Bing 博客上宣布将为 Bing Chat 设置对话次数上限。每个对话只能提问 5 次，每天最多能提问的数量为 50 次。

微软表示：与 Bing Chat 进行长对话会导致 Bing Chat 无法提供符合提问“主题”的准确回答，甚至是错误回答。

该项措施对 99% 的用户没有影响，根据微软的数据，大约只有 1% 左右的用户会每天向 Bing Chat 提问超过 50 次。
