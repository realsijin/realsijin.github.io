Tiny 11：占用仅 8GB 的极简版 Windows 11

作者：realsijin | 发表于： 2023-02-14

Tiny 11 是一款超精简的第三方 Windows 11 系统，目前最新版本基于 Windows 11 22H2，最低支持 2GB RAM + 8GB 存储的设备运行。老款的 Windows 10 设备完全可以直接安装 Tiny 11。

Tiny 11 的其他优点：

  • Windows 11 的必要功能都得到了保留，如：Microsoft Store、Windows Terminal 等
  • 没有预装 Edge 浏览器（现在变得越来越臃肿）
  • 默认支持离线账号（同时也支持连接 Microsoft 账号）

Tiny 11 的一些风险：

  • 存在一些隐私漏洞（其实也无需太过担心）
  • 不支持 Windows Update 后续系统大版本更新（或许有人会喜欢这点）

有想法让老机器换上新装的，不妨试试 Tiny 11。镜像地址：https://archive.org/details/tiny-11_202302
